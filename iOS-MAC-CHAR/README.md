## Set up
#### Server running
```bash
# in your empty directroy
$ git clone https://github.com/NHNNEXT/2017-01-HUDI-MAC-CHAR
$ cd WEB-MAC-CHAR
$ npm install
$ npm run dev
$ ./gradlew build
$ ./gradlew bootRun
```
and, enter http://localhost:8080/ on browser.
